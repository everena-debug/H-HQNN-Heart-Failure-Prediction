# H-HQNN: Hybrid Quantum-Classical Neural Network for Heart Failure Prediction

This repository contains the complete implementation of the H-HQNN framework proposed for heart failure prediction using heuristic feature selection and hybrid quantum-classical neural networks.

## Repository Contents

- Data preprocessing
- Exploratory data analysis
- Feature selection
- Particle Swarm Optimization
- Quantum feature encoding
- Hybrid Quantum Neural Network
- Model training
- Model evaluation
- ROC analysis
- Confusion matrix
- Performance metrics

## Dataset

Heart Failure Clinical Records Dataset

https://archive.ics.uci.edu/ml/datasets/Heart+failure+clinical+records

or

https://www.kaggle.com/datasets/andrewmvd/heart-failure-clinical-data
